What is === operator?

Ans.

This Operator is known as Strict equality.whether its two operands are equal, returning a Boolean result.
and operands value and datatype both are same at that time it will generated true(boolean result).

JavaScript Demo: Expressions - Strict equality operator

console.log(1 === 1);
// Expected output: true (both value are same and datatype(number) is also same)

console.log('hello' === 'hello');
// Expected output: true (both value are same and datatype(string) is also same)

console.log('1' === 1);
// Expected output: false (both value are same but datatype is different that's why result will be false)

console.log(0 === false);
// Expected output: false (both value is different and one value type is number and another is string that's why result will be false)
